# BinaApps — Associate Management System (AMS)

**Project Owner:** Bilal Dwi Nugraha (CEO)
**Project Lead:** Faisal Alfarizi (IT Lead)
**Status:** Sprint 0 — Foundation

## Vision

Membangun fondasi digital BinaHub. AMS adalah produk pertama dari inisiatif BinaApps.

## Sprint 1 Goals

- Associate Portal: registrasi, login, kelola profil, upload CV
- Admin Dashboard: lihat daftar associate, cari, buka profil, download CV standar

## Prinsip

- **Input Once. Use Everywhere.**
- Hindari input data berulang
- Database sebagai single source of truth
- AI untuk otomatisasi, bukan menggantikan keputusan manusia
- Utamakan UX untuk Associate dan Admin
